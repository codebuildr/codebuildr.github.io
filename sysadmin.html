1. Working with BASH arrays and command substitution: a. Declaration of the BASH array array_name=("value1" "value2" "value3" "value4") b. Access the array elements echo ${array_name[0]} echo ${array_name[1]} echo ${array_name[2]} echo ${array_name[3]} c. Print all values at one time echo ${array_name[*]} value1 value2 value3 value4 d. Get array length echo ${#array_name[@]} e. Use arrays values for command arguments (using backticks) mkdir `echo ${array_name[*]}`
